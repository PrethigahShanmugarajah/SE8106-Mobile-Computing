## 📘 Activity 01

### (01) Sum of Even Numbers

Write a program that takes a list of integers and prints the sum of all even numbers in the list.

### (02) Maximum in a List

Write a program to find the largest number in a list of integers without using the `max()` function.

### (03) FizzBuzz

Print numbers from 1 to 50.

- For multiples of 3, print **"Fizz"**
- For multiples of 5, print **"Buzz"**
- For multiples of both 3 and 5, print **"FizzBuzz"**

### (04) Count Vowels in a String

Given a string, count how many vowels (a, e, i, o, u) it contains.

### (05) Remove Duplicates from a List

Given a list of integers, create a new list that contains the same elements without duplicates.

### (06) Reverse a List

Write a program that reverses a list of strings without using the built-in `reversed()` function.

### (07) Factorial of a Number

Write a program that calculates the factorial of a given number using a `for` loop.

### (08) Sum of Positive Numbers

Given a list of integers, print the sum of only the positive numbers.

### (09) Number Guessing with While Loop

Write a program where the user has to guess a number between 1 and 10. Keep asking until the correct number is guessed.

### (10) Print a Pattern

Print the following pattern using nested loops:

```
*
**
***
****
*****
```
